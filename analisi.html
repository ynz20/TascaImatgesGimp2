Aspectes a Millorar

  • Compressió d'Imatges

    Serveix imatges en formats de pròxima generació com WebP per reduir la mida sense perdre qualitat (estalvi potencial de 478 KiB). Redimensionar imatges adequadament pot estalviar 159 KiB.

  • Declaració Explícita de Dimensions

    Afegir atributs width i height a les etiquetes d'imatges evita moviments de contingut durant el carregament.

  • Eliminació de Recursos Bloquejants

    Identificar i eliminar recursos que bloquegen la renderització pot estalviar 340 ms.

  • Minimització i Eliminació de JavaScript

    Minificar el codi JavaScript pot estalviar 42 KiB. Eliminar JavaScript no utilitzat pot estalviar fins a 1,130 KiB.

  • Meta Descripció

    Afegir una meta descripció millora com es mostra la pàgina als resultats de cerca.

  • Ordre dels Encapçalaments

    Mantenir un ordre jeràrquic correcte entre els encapçalaments (H1, H2, H3...).

  • Contrast de Colors

    Ajustar colors de fons i text per complir amb les normes mínimes de contrast (4.5:1).

  • Accessibilitat dels Botons

    Proporcionar noms accessibles als botons mitjançant aria-label o text visible.
